# netflix-rank-fetcher

An app that allows users to enter a list of Netflix shows, separated by either comma or newline and the app fetches series / movies info from OMDb API and displays only the rating of given list, ordered by the rank, descending (in a simple table)

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/netflix-rank-fetcher.git
cd netflix-rank-fetcher
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
